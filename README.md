
# Запчастица

### Приложение предоставляет вам расценки на запчасти

Подготовьте и загрузите .xlsx файл, где в первой колонке - артикулы, а во второй - количество запчастей.
В ответ вам будет предложен файл с ценами.

##### Установка

1 Поместить директорию infra на сервер
2 Дополнить .env файл своими данными, где указано
3 Выполнить docker compose up
